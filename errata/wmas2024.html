3.5 Media specifications

  • Web Audio API – Bibliography link text is now "WEBAUDIO-1.0" instead of just "WEBAUDIO".
  • "WebRTC 1.0: Real-Time Communication Between Browsers" is renamed to "WebRTC: Real-Time Communication in Browsers"

3.6 Graphics specifications

  • Remove "HTML Canvas 2D Context [2DCONTEXT]"
  • Add this bullet point as the last item in the list:
    • "Note: HTML Canvas 2D Context is also required as part of the HTML specification [HTML]."

3.9 Security specifications

  • Add this bullet point to be the 2nd bullet:
    • "HTTP Strict Transport Security (HSTS) [RFC6797]"

3.11 Other web specifications

  • "Indexed Database API [INDEXEDDB]" should be changed to "Indexed Database API 2.0 [INDEXEDDB-2]"

5.3 WebAssembly specifications

Replace all the content with the following four bullets:

  • WebAssembly Core Specification [WASM-CORE-2]
  • WebAssembly JavaScript Interface [WASM-JS-API-2]
  • WebAssembly Web API [WASM-WEB-API-2]
  • Note: In addition to the features specified in the 2.0 versions of the WebAssembly specifications, Extended Constant Expressions (as defined by the WebAssembly Specification Release 3.0 Draft) is also expected to be included given its support across all four of the most widely used user agent code bases.
